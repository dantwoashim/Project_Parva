# Third-Party Notices

## Swiss Ephemeris / pyswisseph

Project Parva depends on `pyswisseph`, the Python bindings for Swiss
Ephemeris, for astronomical calculations used across the API and frontend.

- Upstream package: `pyswisseph`
- Upstream project: Swiss Ephemeris / Astrodienst
- Role in this project: ephemeris-backed tithi, panchanga, kundali, muhurta,
  and festival calculations
- Deployment posture for this repository: AGPL-compatible zero-budget path

If you deploy a hosted service based on this repository, publish the
corresponding source for the exact running version and set `PARVA_SOURCE_URL`
to that location so users can access it from the service itself.
