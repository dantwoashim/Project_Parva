# Commercial Offering

Project Parva is open source. The repository provides the public core.

Separate commercial work may include:

- managed hosting
- integration support
- custom data or partner overlays
- operational support or SLAs
- implementation help for widgets, embeds, or internal workflows

What is not included in the open-source guarantee:

- guaranteed uptime
- private support
- custom feature delivery
- proprietary partner datasets or admin tooling unless they are explicitly added to the public repo
