# Parva Temporal Spec 1.0: Release Note

Today we are publishing **Parva Temporal Spec 1.0**, an open specification for verifiable multi-calendar computation.

## Highlights
- Stable v3 API contract (`/v3/api/*`)
- Deterministic explainability traces (`/v3/api/explain/{trace_id}`)
- Confidence + uncertainty metadata in key date responses
- Provenance and Merkle proof endpoints for auditability
- Conformance runner and benchmark packs for independent verification
- Institutional offline bundle and parity verification scripts

Parva moves ritual-time computation from static, opaque date tables to transparent and reproducible infrastructure, enabling institutions, developers, and communities to verify exactly how dates were computed.
