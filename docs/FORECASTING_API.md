# Forecasting API (Year 3 Week 13-16)

## Endpoints
- `GET /api/forecast/festivals?year=2050&festivals=dashain,tihar`
- `GET /api/forecast/error-curve?start_year=2030&end_year=2050`
- Versioned aliases are available under `/v2/api/forecast/*`.

## Response Highlights
- `horizon_years`: distance from current year.
- `estimated_accuracy`: confidence decay estimate in `[0,1]`.
- `confidence_interval_days`: widening interval for long-horizon planning.

## Notes
- Forecast results are produced by the v2 festival engine.
- Metadata is for planning UX; downstream apps should display confidence decay clearly.
